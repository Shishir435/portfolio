# portfolio

Personal portfolio built with [Astro](https://astro.build) and [Starlight](https://starlight.astro.build).

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [localhost:4321](http://localhost:4321).

## Build

```bash
pnpm build
pnpm preview
```

Blog posts are fetched from [obsidianNotes](https://github.com/Shishir435/obsidianNotes) on build via `scripts/fetch-blog-posts.mjs`.
